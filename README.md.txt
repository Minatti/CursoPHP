CursoPHP

Olá, este repositório é de um curso completo sobre a linguagem PHP, na versão mais atualizada (7*), curso disponibilizado no site: udemy.com.br da empresa Hcode.

Parte do nível básico ao mais avançado com o objetivo de criar um projeto completo. Mesmo que você não tenha conhecimento algum e nunca tenha programado, conseguirá acompanhar com inúmeros exemplos e dicas dos Instrutores que atuam a mais de 10 anos com PHP.

É importante ter o conhecimento básico sobre os assuntos: HTML5, CSS e Lógica de Programação.
